# 班级随机抽签微信小程序
## 项目说明
本项目为大学计算机课程自主实践作业，基于原生微信小程序开发，实现课堂随机点名、小组抽签功能，操作简单，适配手机端。

## 技术栈
原生微信小程序 WXML + WXSS + JavaScript
本地缓存存储名单（无云开发简易版）/ 微信云开发数据库（完整版）

## 核心功能
1. 批量录入班级学生名单，支持一键清空、单独删除人员
2. 抽签滚动动画，抽取后自动剔除已选中人员，不重复抽签
3. 自定义单次抽取人数，支持重置抽签池
4. 本地持久化存储名单，再次打开无需重复输入
5. 记录每一轮抽签历史，可查看、清空记录
6. 简约校园UI界面，适配各类手机尺寸

## 运行步骤
1. 使用微信开发者工具导入本项目文件夹
2. 填入自己的小程序 AppID（测试可使用测试号）
3. 点击编译即可预览，支持真机调试

## 项目用途
课堂随机点名、活动抽奖、班级随机分组
